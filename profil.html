    Profil précis de Chris Coyier

    Chris Coyier est un web developpeur et un web designer. Il a construit le site CSS-Tricks

    Il est aussi le co-fondateur de CodePen, un environnement de développement social pour les concepteurs et développeurs Web front-end. Il permet aux utilisateurs d'écrire du HTML, du CSS, du JavaScript et d'autres langages associés.

    Avec Dave Rupert, Chris est le co-animateur d'un podcast appelé ShopTalk, une émission sur, la création de sites Web.

    Chris a pris la parole lors d'événements et a donné des ateliers dans le monde entier lors de conférences.
