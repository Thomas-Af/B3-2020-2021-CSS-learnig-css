Page d'accueil :

Pour le design de la page d'accueil j'ai utilisé l'élément CSS "Columns" afin de repartir mes différents contenus. Il n'y avait pas beaucoup de contenu c'est pourquoi j'ai utilisé l'attribut "Columns"

Page profil :

Pour la page profil je n'ai pas utilisé de mise en page spécifique, excepté pour les photos ou j'ai utilisé les attributs "object-fit" et "picture" afin de rendre les images disponibles pour tous les écrans.

Page personnalité :

La page personnalité est faite avec "Flexbox". J'aurais pu utiliser "Css Grid" pour la mise en page mais en ayant déjà utilisé "Grid" sur une autre page j'ai préféré utiliser "Flexbox", afin de donner l'aspect qu'à la page actuellement, surtout que mon contenu permettait une mise en page correcte.

Page évolution :

Le début de la page évolution est fait normalement sans attributs spécifique tandis que la seconde partie est faite avec "CSS Grid" pour la mise en pages des propriétés marquantes, afin de faire une coupure avec le texte du début de pages et afin de structure la seconde partie.

Page mentions :

Pour cette page qui est la page mentions légales, j'ai utilisé la propriétés "CSS Grid" pour structurer au mieux cette page, car comme j'avais des parties bien distinctes je me suis dit que l'attribut "CSS Grid" serait parfait pour cette page.

Animations :

J'ai aussi utilisé des animations sur mon site afin de le rendre plus dynamique. J'ai utilisé des animations comme "keyframes" ou "transition".
