Page d'accueil :

Pour le design de la page d'accueil j'ai utilisé l'élément CSS "Columns" afin de repartir mes différents contenus. Il n'y avait pas beaucoup de contenu c'est pourquoi j'ai utilisé l'attribut "Columns"

Page profil :

Pour la page profil j'ai utilisé je n'ai pas utilisé de mise en page spécifique, excepté pour les photos ou j'ai utilisé les attributs "object-fit" et "picture".

Page personnalité :

La page personnalité est faite avec "Flexbox". J'aurais pu utiliser "Css Grid" pour la mise en page mais en ayant déjà utilisé "Grid" sur une autre page j'ai préféré utiliser "Flexbox".

Page évolution :

Le début de la page évolution est fait normalement sans attributs spécifique tandis que la seconde partie est faite avec "CSS Grid" pour la mise en pages des propriétés marquantes.
